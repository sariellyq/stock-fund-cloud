# 基金盘中估算工具 - 云端版 v2.0

## v2.0 功能精简

### 股票部分
- 只查看当日5分钟K
- 显示今日涨跌幅、最新价、前收
- 可手动新增股票记录
- 可维护股票代码列表

### 基金部分
- 先创建基金记录
- 输入基金持有金额
- 给基金添加重仓股票
- 添加重仓时从股票列表中过滤选择股票，并输入占比
- 按重仓股票当前今日涨跌幅估算基金涨跌和盈亏

### 配置保存
Streamlit免费云端不保证长期保存会话状态，所以增加：
- 下载当前配置JSON
- 导入配置JSON

这样你设置好基金和股票后，可以下载配置文件。下次打开云端网页时再导入即可恢复。
