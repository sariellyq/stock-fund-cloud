# 基金盘中估算工具 V4.2 - 行业分布下拉添加版

## V4.2 修改

基金行业分布配置改成和“添加重仓股票”类似的下拉添加形式：

- 从“行业ETF管理”的行业列表中选择行业
- 输入行业占比
- 点击“添加/更新行业占比”
- 已添加行业可以单独编辑或删除

这样新增“资源”等行业后，不会把基金配置面板撑得很长，手机上更好用。

## 仍然保留

- Supabase 长期保存
- 动态行业
- 保守估算看板
- 行业补充估算看板
- 股票当日5分钟K
- 基金盘中5分钟估算曲线

## 更新步骤

1. 覆盖 GitHub：
   - app.py
   - requirements.txt
   - README.md
   - supabase_schema.sql

2. Supabase SQL Editor 运行新的 supabase_schema.sql  
   如果你已经运行过 V4.1 的 schema，也可以重复运行，不会影响已有数据。
