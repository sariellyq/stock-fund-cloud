# 基金盘中估算工具 - 云端版 v3.0

## v3.0 变化

### 估算逻辑
- 已知重仓股票：按当前今日涨跌幅 × 持仓占比计算贡献
- 现金：默认0涨跌
- 其他资产：可手动输入今日估算涨跌幅
- 未知仓位：保守估算中不计算，按0处理

### 显示内容
- 已知股票占比
- 现金占比
- 其他资产占比
- 未知仓位
- 保守估算涨跌幅与盈亏
- 推演估算涨跌幅与盈亏
- 基金盘中5分钟估算走势

### 关于5分钟走势
开盘后刷新，程序会抓取当日5分钟K：
- 每个股票每5分钟计算一次今日涨跌幅
- 再按重仓占比加权
- 得到基金盘中每5分钟的估算涨跌曲线

注意：Yahoo Finance 可能有延迟，不保证严格实时，只适合盘中粗略估算。
